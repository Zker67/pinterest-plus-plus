<h2>Features</h2>

<ul>
<li>Adds polished hover buttons to the top-right corner of Pinterest grid cards.</li>
<li>The download button prioritizes Pinterest PinResource data to resolve the original media URL.</li>
<li>Supports image and video Pins. Video Pins prefer directly downloadable MP4 sources.</li>
<li>The star button delegates to Pinterest's native save button and reflects the official saved state.</li>
<li>Adds a matching download button beside the native save button on Pin detail pages.</li>
<li>On detail pages, failed original downloads fall back to the currently displayed media to avoid AccessDenied pages.</li>
</ul>

<h2>Usage</h2>

<ul>
<li>Hover an image or video card in the Pinterest grid, then click the top-right download icon.</li>
<li>Click the star icon to save the Pin. The state follows Pinterest's native button.</li>
<li>On a Pin detail page, click the download icon beside the save button.</li>
</ul>

<h2>Notes</h2>

<p>This script is based on TiLied's Pinterest Plus. The old Full Size preview flow was removed and replaced with download/save controls that fit Pinterest's native UI more closely.</p>
